# Deep Learning Projects

This repository contains a collection of deep learning projects implemented in **PyTorch**, covering computer vision, natural language processing, transfer learning, and image captioning. Each project demonstrates the implementation of different neural network architectures and deep learning techniques.

## Projects

### 1. Rice Variety Classification
- Classification of rice varieties using a fully connected neural network (ANN).
- **Techniques:** Feedforward Neural Networks, Classification

### 2. Animal Faces Classification
- Image classification of animal faces using a Convolutional Neural Network (CNN).
- **Techniques:** Convolutional Neural Networks, Computer Vision

### 3. Bean Leaf Lesion Classification
- Bean leaf disease classification using transfer learning with a fine-tuned GoogleNet model.
- **Techniques:** Transfer Learning, GoogleNet, Fine-Tuning

### 4. Image Captioning
- Automatic image caption generation using a CNN encoder and an LSTM decoder.
- **Techniques:** CNN, LSTM, Encoder–Decoder Architecture

### 5. BBC Text Classification
- News article classification using a Vanilla Recurrent Neural Network (RNN).
- **Techniques:** Natural Language Processing, RNN

## Technologies

- Python
- PyTorch
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Repository Structure

Each project is provided as a standalone Jupyter notebook, along with any associated trained model checkpoints.

## Learning Objectives

- Build deep learning models from scratch using PyTorch.
- Apply convolutional and recurrent neural networks to real-world datasets.
- Explore transfer learning for image classification.
- Compare different architectures across computer vision and NLP tasks.
